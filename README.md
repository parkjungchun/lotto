# 🍀 통계 기반 로또 번호 생성기 (Lotto Number Generator)

사용자의 브라우저에서 직접 실행되는 가벼운 웹 기반 로또 번호 생성기입니다. 단순히 무작위 번호를 추출하는 것이 아니라, 역대 당첨 데이터를 바탕으로 통계적 확률이 높은 구간 내의 번호만을 필터링하여 생성합니다.

## 🚀 주요 기능

* **통계적 필터링**: 무의미한 번호 조합을 배제하고 당첨 가능성이 높은 최적의 조합을 생성합니다.
* **주 1회 생성 제한**: 실제 로또 구매 주기와 맞춰 매주 일요일을 기점으로 주 1회만 번호를 생성할 수 있도록 관리합니다.
* **데이터 보존**: `localStorage`를 활용하여 브라우저를 새로고침하거나 닫아도 이번 주에 생성한 번호를 유지합니다.
* **초기화 기능**: 필요한 경우 저장된 데이터를 삭제하고 번호를 다시 생성할 수 있습니다.
* **반응형 UI**: 모바일 환경에서도 최적화된 화면 구성을 제공합니다.

## 📊 적용된 필터링 로직 (Algorithm)

본 생성기는 전체 약 814만 개의 조합 중, 역대 당첨 회차(최근 1222회 등)에서 공통적으로 나타나는 특징을 반영하여 약 **4,341,577개(전체의 53.3%)**의 유효 조합만을 추출합니다.

1.  **구간 분산 (Range Distribution)**
    * 번호가 특정 구간에 쏠리지 않도록 1~15, 16~30, 31~45 구간에서 최소 1개 이상의 번호가 포함되도록 설계되었습니다.
2.  **홀짝 균형 (Odd/Even Balance)**
    * 극단적인 홀수나 짝수만 나오는 경우를 배제하기 위해 홀수의 개수를 2개에서 4개 사이로 유지합니다.
3.  **합계 제한 (Sum Range)**
    * 6개 번호의 총합이 통계적으로 가장 빈번하게 발생하는 **100 ~ 170** 사이의 조합만 생성합니다.

## 🛠 기술 스택

* **Frontend**: HTML5, CSS3 (Google Fonts - Black Han Sans, Noto Sans KR)
* **Script**: Vanilla JavaScript
* **Storage**: Browser LocalStorage

## 📝 사용 방법

1.  `lotto.html` 파일을 웹 브라우저로 실행합니다.
2.  **[번호 생성하기]** 버튼을 클릭하여 이번 주의 행운 번호 5세트를 확인합니다.
3.  이미 생성한 경우, 다음 주 일요일 00시까지 버튼이 비활성화되며 다음 생성 가능 시간을 안내합니다.
4.  새로운 번호를 뽑고 싶다면 **[데이터 초기화]** 버튼을 눌러 기록을 지울 수 있습니다.

## ⚠️ 주의사항

* 본 프로그램은 통계적 확률을 기반으로 번호를 추천할 뿐, **당첨을 보장하지 않습니다.**
* 모든 결과는 사용자의 로컬 환경(브라우저)에 저장되며, 외부 서버로 데이터를 전송하지 않습니다.
